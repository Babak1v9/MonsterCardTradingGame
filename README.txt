# SWE Projekt Lukas Rittler

## Datenbank

Der Server verwendet eine lokale PostgreSQL Datenbank. Die Datenbank muss mit dem "create.sql" Skript aufgesetzt werden, das unter: 

exam-Babak1v9/Database/create.sql

gefunden werden kann.

Die Datenbank muss weiters auf localhost (127.0.0.1) auf dem Port 5432 hören. Als Datenbankname muss "SWE" gesetzt werden und der User muss folgende Login Daten haben:

Benutzername: postgres
Passwort: swe


## Starten des MTCG-Servers

Der Server kann mithilfe der "MyServer.exe" gestartet werden zu finden unter:

exam-Babak1v9/MyServer.exe 
oder unter:
exam-Babak1v9/dotnet/swe1/MyServer/bin/Debug/netcoreapp3.1/MyServer.exe


## Zeitaufwand

Insgesamt habe ich für das Projekt ca. 8 Tage mit jeweils geschätzten 7-9 Stunden gebraucht, dies entspricht einem Gesamtaufwand von ungefähr ~ 80 Stunden.
